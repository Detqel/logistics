Folders in Zip File

  1. css — folder with CSS and SCSS files.
  2. fonts — folder with fonts files.
  3. images — folder with images files.
  4. js — folder with Javascript files.
HTML Files:
  1. index.html — Home page 1
  2. index-2.html — Home page 2
  3. about-1.html — About 1 page
  4. blog-grid.html — Blog grid page
  5. blog-single.html — Blog detail page
  6. contact.html — Contact page
  7. services-1.html — Services style 1 page
